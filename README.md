# OdontoGate — Backend

Backend del proyecto **OdontoGate**, desarrollado con Java, Springboot y Hibernate.

---

## Requisitos previos

- **Java 24** 
- **Git** 
- **Docker Desktop** 

---

## Configuración inicial

### 1. Descarga el script de configuración

El script se encarga de clonar el repositorio, instalar dependencias y lanzar el servidor automáticamente.

**Windows:**

Descarga y ejecuta [`setup.bat`](./setup.bat)

```bat
setup.bat
```

**Linux / macOS:**

Descarga y ejecuta [`setup.sh`](./setup.sh)

```bash
bash setup.sh
```

---

### 2. Antes de ejecutar el script

> ⚠️ **Importante:** Ten en cuenta lo siguiente antes de correr el setup:

- **Docker Desktop debe estar abierto y corriendo** antes de ejecutar el script.
- **El puerto `8080` debe estar libre** — asegúrate de que ninguna otra aplicación lo esté usando.
